# 🎓 College ERP System

A full-stack College ERP (Enterprise Resource Planning) system built using Spring Boot and React JS to manage academic and administrative operations efficiently.

---

## 🚀 Tech Stack

### Backend
- Java
- Spring Boot
- Spring Data JPA
- MySQL

### Frontend
- React JS
- Axios
- Bootstrap / CSS

---

## 📌 Features

- Student Management
- Faculty Management
- Course Management
- Authentication & Authorization
- RESTful APIs
- CRUD Operations
- Database Integration using JPA & MySQL

---

## 🏗️ System Architecture

Frontend (React JS) → REST APIs → Spring Boot → MySQL Database
